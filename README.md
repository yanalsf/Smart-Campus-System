# Smart Campus Activity System 

This project demonstrates the evolution of a Java-based registration system through different programming paradigms.

## Project Structure
- **/1_Procedural_Code**: The initial version built using a procedural approach, focusing on functions and sequential logic.
- **/2_OOP_Code**: The refactored version implementing Object-Oriented Programming (OOP) principles (Encapsulation, Inheritance, Classes).
- **/3_Event_Driven_Code**: The final version featuring a Graphical User Interface (GUI) that handles user interactions through events.

## Key Skills Demonstrated
- Java Programming
- Object-Oriented Design (OOP)
- GUI Development (Swing/JavaFX)
- Architectural Refactoring
